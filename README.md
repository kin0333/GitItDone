# Git It Done

A project management tool to help you organize and track your tasks efficiently.

## Features

- Task creation and management
- Priority levels
- Due date tracking
- Progress monitoring

## Installation

```bash
git clone https://github.com/kin0333/GitItDone.git
cd GitItDone
```

## Usage

```bash
npm install
npm start
```
